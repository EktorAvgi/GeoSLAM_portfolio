Installing/Activating/Deactivating of the Software License.


Installation of the License

The installation process is very simple and can be found in the user manual. The user just need to run the executable bundle add the license number that is been given in a document as you may see below.


Activating & Deactivating the License

This process is for enabling different machines which are on different geo-graphic locations to process data-sets. It is very important for the users to de-activate initially the license from one machine (machine A) in order to activate it on another (machine B). On any other case, please contact support to return the license.
